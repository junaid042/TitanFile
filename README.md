
# 📤 TitanFile Outlook Add-in (Secure File Upload & Sharing)

TitanFile Outlook Add-in enables users to securely upload and share files via TitanFile directly from Outlook. It replaces traditional email attachments with secure file transfer, ensuring compliance, encryption, and safe communication.

## ⚙️ How It Works

1. Compose a new email in Outlook  
2. Launch the TitanFile Add-in  
3. Upload files securely to TitanFile  
4. Generate a secure file-sharing link  
5. Insert the link into the email  
6. Send email with secure access instead of attachments  

---

### 5. Sideload Add-in into Outlook

* Open Outlook
* Go to Add-ins → Upload My Add-in
* Upload the manifest file
